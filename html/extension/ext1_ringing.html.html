扩展任务一：低通滤波器振铃效应对比

对比理想低通、巴特沃斯低通、高斯低通三种滤波器的振铃效应，分析频域截断特性与空域伪影的关系

1 问题建模：三种低通滤波器的数学定义

振铃效应的根源在于频域的陡峭截断。不同滤波器对高频的衰减方式决定了空域是否产生虚假波纹。

理想低通 (ILPF)

H(u,v) = { 1, D≤D₀ ; 0, D>D₀ }

频域硬截断，对应空域sinc函数卷积 → 振铃严重

巴特沃斯低通 (BLPF)

H(u,v) = 1 / [1 + (D/D₀)2n]

n阶平滑过渡，n越大越接近理想 → 振铃可控

高斯低通 (GLPF)

H(u,v) = exp[ −D² / (2·D₀²) ]

完全平滑衰减，无旁瓣 → 无振铃

吉布斯现象：当频域存在不连续点（理想低通的硬截断），其傅里叶逆变换（空域冲激响应）会产生无限振荡的sinc函数旁瓣。 这些旁瓣在卷积时会在图像边缘附近产生虚假的明暗波纹——即振铃效应。 巴特沃斯滤波器的平滑过渡可抑制旁瓣幅度，高斯滤波器因无限光滑而完全消除振铃。
